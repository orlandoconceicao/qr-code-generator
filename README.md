# 📱 QR Code Generator

Projeto de interface para geração de **QR Codes**, desenvolvido para praticar **HTML, CSS e JavaScript com manipulação de DOM**.

A aplicação permite que o usuário gere QR Codes dinamicamente a partir de **URLs ou textos personalizados**.

---

# 🚀 Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
- Manipulação do DOM
- Eventos de clique e teclado
- API de geração de QR Code
- Design responsivo

---

A aplicação possui as seguintes funcionalidades:

✔ Geração dinâmica de QR Code

✔ Atualização automática da imagem
